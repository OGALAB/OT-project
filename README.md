# OT-project
AIを技術メンターとして、フロントエンド技術の基礎からモダンなフレームワークまでの徹底習得と、実践的な設計・開発・運用を行っていきます。

----

## 目的
- **AIをメンターとした効率的な学習**
  エラーの解決能力向上
  コードの設計・リファクタリング
  技術のインプット＆アウトプット

- **モダンフロントエンドスタック習得**
  HTML/CSS基礎からReactを用いた実践的なWebツール（アプリ）開発

- **プロダクトの自作・リリース**
  自己学習で終わらせず、実際のプロジェクトを立ち上げて企画・実装・公開

----

## 技術スタック
- **現在学習・利用中**
  HTML、CSS、JavaScript、TypeScript、React

- **今後学習・導入予定**
  Next.js

- **メンター**
  Gemini、ChatGPT

- **ツール**
  VSCode、Git、GitHub、Vercel

----

## 学習＆開発ロードマップ
1. **基礎学習**
- **HTML、CSS**
  レスポンシブデザイン等
- **JavaScript、TypeScript**
  DOM操作、データ処理、条件分岐、繰り返し、関数、非同期処理
  型定義、Interfaceの基礎

2. **モダンフロントエンドの学習**
- **React**
  JSX、TSX、コンポーネント管理、Props、State
  useState等のHooks

3. **実践的挑戦**
- **プロダクト開発・立ち上げ**
  企画・要件定義、アイディア出し

- **Next.js**
  今後学習していきたい

----

## AIメンターとの動き方
学習・開発を効果的に進めるため、AIメンターに対して以下の役割を依頼してます。

1. **コードレビュー**
   作成したコードの可読性・パフォーマンス等の視点でフィードバックを貰う
2. **エラー解決**
   単に答えを聞くのでは無く「なぜ起きたか」「どう調べるべきか」を話し合う
3. **ステップアップの相談**
   ReactからNext.jsへ移行するタイミングや、学習ロードマップの調整に関するアドバイスを貰う
4. **言語化トレーニング**
   実装方針について話し合い、技術的な意思理由等を言語化する
